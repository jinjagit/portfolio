Portfolio


Dream Flights

Dream Flights

A proof-of-concept Rails application. The focus of this exercise was practice in creation / usage of advanced forms, and specifically forms which submit nested attributes. The core function is the creation of flight bookings, which include basic passenger details.

Social Light

Social Light

A proof-of-concept Rails application. The focus of this exercise was practice in creating a through-table (Attendances) and the appropriate creation / usage of related Active Record associations (User <=> Attendance(s) <=> Event). The core function is the creation of events which include a list of attendees (invited users), and the filtering of such events by date (future vs. past) when shown on the User page (named: 'events').

Ruby Chess

Ruby Chess

A graphical chess interface, written in pure Ruby and making use of the Ruby2D gem. All rules of chess are enforced, including all legal moves, forced draws, draw through repetition, etc. The GUI is intuitive and offers many optional aids to the user, such as highlighting of legal moves, algebraic notation and the ability to review previous moves. Games can be saved and loaded, and saved incomplete games can be continued.

Calculator

Calculator

A mobile focused, minimal, responsive design, with desktop default layout (scaled relative to screen size and responsive under certain conditions), written in pure JavaScript. The calculator offers parenthesized expressions and exponentiation, as well as the normal basic operations. Various error messages are displayed when appropriate. Scientific notation is used to enable display of very large / small values. The overall design is clean and intuitive.
